  • Please accept terms & condition

    If no HW ID, need apply a new account by real email or mobile phone number firstly

Join a BETA campaign:

  • Start APP
  • login
  • select “Me”
  • “Join Project”
  • “Available Projects”

BETA version HOTA:

you will receive HOTA upgrade notification automatically, then you can select to update your phone.

Problem feedback:

during the test time, if you find any problem you can go to “Report” page, and select relevant module which your problem related with, then input all necessary information and submit

  • Go to ”Report” page, select the module which has problem
  • Input detail info about the problem you found and then submit
  • PS: when you submit the problem, the APP will ask you if you agree to upload log when you are not using WIFI

Quit the BETA:

  • Start APP
  • login
  • select “Me”
  • “Join Project”
